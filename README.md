# Abdesalam & Hams — Wedding Invitation

A React recreation of the elegant digital wedding invitation from the reference video (`monicacarlos.netlify.app`).

## Features

- **Envelope intro** — Arabic welcome text with animated envelope open on click
- **Save the Date** — Live countdown to 10 July 2026
- **Our Love Story** — Vertical timeline of relationship milestones
- **Our Memories** — Horizontally scrollable photo gallery
- **Event Details** — Date, time, and venue cards with map link
- **Mobile-first** — Dark navy & gold theme optimized for phones

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

## Customize

Edit `src/data/content.ts` to change names, dates, timeline events, photos, and venue details.

Replace the Unsplash placeholder images in `HeroImage.tsx` and `content.ts` with your own photos.

## Tech Stack

- React 19 + TypeScript
- Vite
- Plain CSS (no extra dependencies)
